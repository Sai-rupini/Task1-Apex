HTML: Structure of the Web

HTML (HyperText Markup Language) is the backbone of every webpage. It defines the structure and content like headings, paragraphs, images, and links.

<h1>Welcome to My Site</h1>
<p>This is a paragraph.</p>
<a href="https://example.com">Visit Example</a>

JavaScript: Adding Interactivity

JavaScript lets you make webpages interactive, respond to user input, and manipulate page content dynamically.

const btn = document.querySelector('button');
btn.addEventListener('click', () => {
  alert('Hello from JavaScript!');
});
